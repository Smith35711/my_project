## 状态定义

| 状态 | 含义 |
|------|------|
| 🆕 新建 | 测试工程师刚提交 |
| 🔍 分析中 | 开发工程师正在定位根因 |
| 🔧 修复中 | 开发工程师正在修复 |
| 🧪 待验证 | 修复完成，测试工程师重新测试 |
| ✔️ 已验证 | 回归测试通过 |
| ❌ 无法复现 | 经确认无法复现 |
| 📌 暂不处理 | 已知问题但不计划修复 |

## 严重级别

| 级别 | 定义 |
|------|------|
| P0 | 阻塞 — 系统无法运行，无规避方法 |
| P1 | 严重 — 核心功能不可用 |
| P2 | 一般 — 非核心功能异常 |
| P3 | 轻微 — 体验问题或改进建议 |
